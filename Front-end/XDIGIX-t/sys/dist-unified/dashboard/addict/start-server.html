The CORS error happens because you're opening the file directly. You need to run a local server.

Method 1: Python (Easiest)

If you have Python installed:

  1. Open Terminal/Command Prompt
  2. Navigate to this folder: 
    cd sys/apps/dashboard/public/addict
  3. Run: 
    python3 -m http.server 8000
    (or python -m http.server 8000 on Windows)
  4. Open browser: http://localhost:8000

Method 2: Node.js (npx serve)

If you have Node.js installed:

  1. Open Terminal/Command Prompt
  2. Navigate to this folder: 
    cd sys/apps/dashboard/public/addict
  3. Run: 
    npx serve .
  4. Open the URL shown in terminal (usually http://localhost:3000)

Method 3: VS Code Live Server

If you use VS Code:

  1. Install "Live Server" extension
  2. Right-click on index.html
  3. Select "Open with Live Server"

Method 4: PHP (if installed)

  1. Open Terminal
  2. Navigate to this folder
  3. Run: 
    php -S localhost:8000
  4. Open: http://localhost:8000

Why this happens: Browsers block requests from file:// protocol for security. A local server provides a proper http:// origin that the API accepts.
